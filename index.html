Armored Security Suite

Zero compromises. Built specifically to protect your cryptographic identity, conversations, and assets from local malware and physical tampering.

100% On-Chain E2E Encryption

All chat messages are wrapped inside fully encrypted transaction payloads directly on the Algorand ledger. No databases to hack, no intermediate servers, and absolute permanence.

Brute-Force Protection

Dynamic lockout defense mechanisms. The application instantly freezes all authentication routes for 1 minute after exactly 3 consecutive incorrect password attempts, protecting your keys from brute-force scripts.

Biometric Security Shield

Integrates system-level BiometricPrompt (Touch ID / Face ID) to authenticate wallet queries. Protects message reading and key exports instantly with hardware security.

Screenshot Hardware Protection

Uses Android's secure layout flag (FLAG_SECURE) to completely block screenshot captures and overlay attacks. Screen recordings of private channels output as pitch black.

SQLCipher Room Database

Local cached messages, contacts directory, and keys are encrypted using a randomized, hardware-tied key in the Keystore and stored under standard AES-256 local database containers.

Zero External Backend Relays

Allo•Alu talks directly to peer addresses over Algorand. There are no proprietary databases, relay channels, or servers to fail, log IPs, sell data, or shut down.

Decentralized Storage Setup

IPFS Media Sharing via Pinata

Because Allo•Alu does not have central servers, we leverage the power of the InterPlanetary File System (IPFS) to let you send pictures. Setting up your secure gateway takes under a minute.

✔ No middleman ever reviews, filters, or censors your media.

Easy 3-Step Integration:

1
Create free Pinata Account

Go to Pinata.cloud and register. No credit card is required.

2
Generate API JWT Key

Create a "New Key" with Admin capabilities or ensure custom Scopes include:

  • • Files ➔ Write
  • • Pinning ➔ pinFileToIPFS & pinning/pinFileToIPFS
3
Paste JWT inside the App

Copy the long JWT token string from Pinata, open Allo•Alu, head to Settings, and paste it. You are ready to share encrypted photos!
