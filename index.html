Why "No Push Notifications"
Is a Critical Security Choice

Standard Web2 communication tools leak massive metadata vectors via Google Firebase (FCM) or Apple Push services. In order to bring you notifications, a central server must maintain continuous logs linking your IP, exact physical location, active state, and device token to your identity.

✗ No persistent background sockets linking device hardware to your IP.
✗ No central servers tracking when you launch, close, or type in the app.
✓ Zero metadata footprint. Direct client-to-ledger querying.

Pre-Flight Balance Protection

Allo•Alu proactively prevents accidental transaction failures. The application integrates an automated balance monitor which immediately warns the user and locks message transmission if the wallet balance drops below 0.101 ALGO (0.1 ALGO Minimum Balance Requirement + gas fee coverage).
